# 企业投资尽职调查Skill

## 技能概述

本技能提供一套完整的企业投资尽职调查工作流程，通过调用entprise-information-mcp MCP工具获取企业多维度数据，结合网络检索补充公开信息，最终生成专业、系统化的投资尽职调查报告。

## 适用场景

- 投资机构进行项目尽职调查
- 金融机构进行贷款前企业评估
- 企业进行并购目标筛选
- 投资个人进行股权投资项目评估
- 需要了解目标企业综合情况的商业场景

---

## 工作流程

### 第一步：企业信息检索

使用 `get_multi_dimensional_info` 工具根据关键词检索目标企业，返回符合条件的企业列表。

**输入参数**：
- keyword: 企业名称关键词

**输出处理**：
- 分析返回的多家企业，第一家企业通常为母公司或核心主体
- 选择第一家企业作为主要尽调对象
- 记录其他关联企业的信息用于交叉验证

### 第二步：批量获取企业详细信息

**重要**：为提高效率，应将独立的数据查询并行执行，一次性调用多个工具。

**并行查询组一（基础信息）**：
```
get_enterprise_info          - 企业基本信息
get_enterprise_gene          - 基因标签信息
get_enterprise_data_discovery - 数据发现信息
get_enterprise_evaluation    - 综合评价信息
```

**并行查询组二（股权治理）**：
```
get_shareholder_info    - 股东及出资信息
get_main_manager_info   - 主要管理人员信息
```

**并行查询组三（财务融资）**：
```
getEntFinanceCapabilityInfoV2 - 企业财务指标，需查询2023年、2024年两年数据
get_financing_history           - 融资历史
```

**并行查询组四（知识产权）**：
```
get_enterprise_patents_v2  - 专利信息
get_software_copyright     - 软件著作权
get_trademark_info         - 商标信息
```

**并行查询组五（合规风险）**：
```
get_business_abnormal       - 经营异常
get_administrative_penalty  - 行政处罚
get_court_execution         - 被执行人信息
get_court_dishonest         - 失信被执行
```

**并行查询组六（经营状况）**：
```
get_enterprise_investment   - 对外投资
get_enterprise_qualification - 资质认证
get_enterprise_branches     - 分支机构
```

### 第三步：补充网络检索

使用 websearch 工具检索以下信息：

1. **企业新闻动态**：产品发布、战略合作、重大事项、中标项目
2. **行业分析**：行业规模、竞争格局、发展趋势、政策动态
3. **可比公司**：同行业上市公司或竞争对手信息
4. **监管政策**：相关行业政策变化

### 第四步：报告撰写

按照本技能规定的报告结构和撰写规范，生成专业、系统化的投资尽职调查报告。

## 报告结构模板

### 报告封面

```
# [目标企业名称]投资尽职调查报告

**机密文件**

---

**报告编号**：DD-YYYY-MM-NNN
**报告日期**：[日期]
**目标企业**：[企业全称]
**报告类型**：投资尽职调查报告
**委托方**：[投资方名称]
```

### 执行摘要（重点章节）

执行摘要是报告的核心，需高度凝练地呈现投资标的的核心信息、投资逻辑和风险提示。

**结构要求**：

#### 投资标的概览
- 企业定位（1-2句话）
- 核心业务（1-2句话）
- 关键财务数据（营收、净利润）
- 重要资质荣誉

#### 核心投资逻辑（分维度论述）

**优势维度**（每个优势独立成段，先总结后展开）：
- 行业赛道优势
- 技术壁垒优势
- 财务表现优势
- 客户资源优势
- 战略协同优势

**表述范式**："第一，[优势主题]。具体表现/数据支撑。意义/影响。"

#### 主要风险因素

**风险维度**（每个风险独立成段）：
- 行业竞争风险
- 技术迭代风险
- 监管合规风险
- 客户集中度风险

**表述范式**："第X，[风险主题]。具体表现。可能影响。"

#### 投资建议

- 综合评级（推荐投资/谨慎投资/不建议投资，A级/B级/C级）
- 核心理由（3-5点）
- 关注事项（2-3点）

## 报告撰写规范

### 语言风格规范

**基本原则**：
- 使用正式书面语言，避免口语化表达
- 客观陈述事实，避免主观臆断和情绪化表述
- 数据引用需标注来源
- 分析需有逻辑支撑，避免空洞表述

**表述范式**：

**优势描述**："第一，[优势主题]。具体表现/数据支撑。意义/影响。"

**风险描述**："第X，[风险主题]。具体表现。可能影响。"

**结论表述**：先总结后展开，结论先行。

### 分析深度要求

**财务分析**：
- 需计算关键财务比率（毛利率、净利润率、ROA、ROE、资产负债率等）
- 需进行同比/环比分析
- 需进行行业对标分析
- 需识别财务风险信号

**竞争分析**：
- 需明确竞争格局（市场参与者、集中度）
- 需分析竞争要素（数据、技术、客户、资质等）
- 需评估竞争优势和劣势

**风险评估**：
- 需分层次评估（高/中/低）
- 需量化风险影响
- 需提出应对措施

### 格式要求

**表格要求**：
- 关键数据必须以表格呈现
- 表格需有标题、表头、数据行
- 数据需标注单位
- 时间序列数据需按时间顺序排列

**章节要求**：
- 章节标题层次分明（章、节、点）
- 每章需有导语说明章节目的
- 重要结论需有数据支撑

**重点标识**：
- 关键数据可加粗
- 重要结论可独立成段
- 风险提示需醒目标注

## 分析框架

### 财务分析框架

**关键财务指标**：

| 指标类别 | 指标名称 | 计算公式 | 分析要点 |
|---------|---------|---------|---------|
| 盈利能力 | 毛利率 | (营收-成本)/营收 | 反映产品附加值 |
| 盈利能力 | 净利润率 | 净利润/营收 | 反映整体盈利能力 |
| 盈利能力 | ROE | 净利润/净资产 | 反映资本回报能力 |
| 盈利能力 | ROA | 净利润/总资产 | 反映资产回报能力 |
| 成长能力 | 营收增长率 | (本期营收-上期营收)/上期营收 | 反映成长速度 |
| 成长能力 | 净利润增长率 | (本期净利润-上期净利润)/上期净利润 | 反映盈利增长 |
| 偿债能力 | 资产负债率 | 负债总额/资产总额 | 反映财务风险 |

**财务健康判断标准**：

| 指标 | 健康区间 | 警戒区间 | 风险区间 |
|-----|---------|---------|---------|
| 净利润率 | >20% | 10%-20% | <10% |
| ROE | >15% | 8%-15% | <8% |
| 资产负债率 | <50% | 50%-70% | >70% |
| 营收增长率 | >20% | 10%-20% | <10% |

### 竞争优势分析框架

**竞争优势评估维度**：

| 竞争优势 | 评估要素 | 可持续性判断 |
|---------|---------|-------------|
| 数据资源优势 | 数据广度、深度、时效性 | 高（积累壁垒） |
| 技术能力优势 | 专利数量、技术壁垒、研发投入 | 高（技术壁垒） |
| 客户资源优势 | 客户质量、粘性、集中度 | 高（转换成本） |
| 资质牌照优势 | 资质等级、稀缺性 | 高（准入壁垒） |
| 团队背景优势 | 核心团队经验、稳定性 | 中（人才流动） |
| 资本优势 | 融资能力、股东背景 | 中（资本可得性） |

### 风险评估框架

**风险等级定义**：

| 风险等级 | 定义 | 表现特征 |
|---------|------|---------|
| 高 | 对企业经营有重大影响 | 可能导致业绩大幅下滑或经营困难 |
| 中 | 对企业经营有一定影响 | 可能导致业绩波动但可控 |
| 低 | 对企业经营影响较小 | 基本不影响正常经营 |

## 数据解读要点

### 企业评价等级含义

| 等级 | 含义 | 综合评分区间 |
|-----|------|-------------|
| AAA | 综合实力最强，风险最低 | 800分以上 |
| AA | 综合实力较强 | 700-800分 |
| A | 综合实力良好 | 600-700分 |
| BBB | 综合实力一般 | 500-600分 |
| BB | 综合实力较弱 | 400-500分 |
| B及以下 | 综合实力弱，风险较高 | 400分以下 |

### 关键风险信号

**高风险信号**（需重点提示）：
- 行政处罚记录（特别是重大处罚）
- 经营异常记录
- 失信被执行记录
- 持续大幅亏损（连续两年以上）
- 资产负债率超过70%
- 高管频繁变动
- 核心技术人员流失
- 核心客户流失

**中等风险信号**（需关注）：
- 被执行人记录
- 应收账款大幅增长
- 存货周转率下降
- 毛利率持续下降
- 客户集中度过高

### 积极信号

**强积极信号**：
- 国家级专精特新"小巨人"企业
- 国家高新技术企业
- 获得知名投资机构投资
- 核心技术专利布局完善
- 营收快速增长（年增长率>30%）
- 净利润率>30%
- 行业龙头地位

**一般积极信号**：
- 省市级专精特新企业
- ISO体系认证
- 客户资源优质
- 管理团队经验丰富

## 注意事项

### 数据采集注意事项

1. **并行查询**：将独立的数据查询并行执行，提高效率
2. **数据验证**：关键数据需交叉验证，确保准确性
3. **数据时效**：注意数据更新时间，重要决策需核实最新数据
4. **数据完整性**：部分敏感信息可能无法获取，需在报告中说明局限性

### 报告撰写注意事项

1. **逻辑清晰**：报告各章节需有明确的逻辑关系
2. **论据充分**：所有分析结论需有数据支撑
3. **风险突出**：风险提示需醒目，便于投资方快速识别
4. **建议具体**：投资建议需具体明确，避免模糊表述

### 质量控制要点

1. **信息完整性**：确保各章节信息完整，无重大遗漏
2. **分析深度**：确保分析有深度，避免浅尝辄止
3. **表述准确性**：确保专业术语使用准确
4. **格式规范性**：确保报告格式统一规范
