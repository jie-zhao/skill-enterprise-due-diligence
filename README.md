# skill-enterprise-due-diligence
专业的企业投资尽职调查技能，提供全面的企业背景调查、风险评估和投资价值分析。

## 📦 包含技能

### enterprise-due-diligence - 企业投资尽职调查

**功能定位**: 使用企业信息MCP工具进行全面的企业背调

**触发场景**:
1. 询问某企业的背景调查、投资评估或尽职报告
2. 需要查询企业的工商信息、股权结构、融资历史、知识产权、财务数据、风险信息
3. 撰写投资尽调报告、企业估值报告或信用评估报告
4. 对目标企业进行全面的风险评估和投资价值分析

**数据来源**:
- 📋 工商注册信息
- 👥 股东及出资信息
- 🏢 对外投资/分支机构
- 💰 财务数据
- ⚖️ 法律风险信息
- 📜 知识产权
- 🏆 资质荣誉
- 📰 招投标信息

## 🚀 安装使用

### 方式一：直接复制

```powershell
# Windows
xcopy /E /I "enterprise-due-diligence" "%USERPROFILE%\.config\opencode\skills\enterprise-due-diligence"
```

### 方式二：符号链接

```powershell
$repoDir = "C:\path\to\opencode-due-diligence"
$skillsDir = "$env:USERPROFILE\.config\opencode\skills"
New-Item -ItemType SymbolicLink -Path "$skillsDir\enterprise-due-diligence" -Target "$repoDir\enterprise-due-diligence"
```

## 📋 使用示例

### 示例 1：企业全面背景调查

```
请对"某某科技有限公司"进行全面的投资尽职调查，包括：
1. 企业基本信息和股权结构
2. 主要股东和对外投资情况
3. 财务数据和经营情况
4. 法律风险和信用记录
5. 知识产权和资质荣誉
```

### 示例 2：投资前风险评估

```
请分析目标公司的投资风险：
- 经营风险（异常经营、行政处罚）
- 法律风险（被执行人、失信信息）
- 税务风险（欠税、重大税收违法）
- 股权风险（股权质押、冻结）
```

### 示例 3：生成尽调报告

```
请为"某某企业"生成一份完整的投资尽职调查报告，包括：
1. 执行摘要
2. 公司概况
3. 股权结构分析
4. 财务分析
5. 风险评估
6. 投资建议
```

## 🔍 调查维度

### 基础信息
- 企业注册信息
- 法定代表人
- 注册资本
- 经营范围
- 成立日期

### 股权结构
- 股东信息
- 出资比例
- 股权变更历史
- 实际控制人

### 经营状况
- 对外投资
- 分支机构
- 员工数量
- 社会保险缴纳

### 财务状况
- 营业收入
- 利润总额
- 资产负债
- 纳税情况

### 风险信息
- 经营异常
- 行政处罚
- 法院执行
- 失信信息
- 欠税信息
- 限制高消费

### 知识产权
- 专利信息
- 软件著作权
- 商标信息
- 作品著作权
- 企业标准

### 资质荣誉
- 企业资质
- 荣誉奖励
- 融资历史

## 📊 尽调报告结构

1. **执行摘要** - 核心发现和投资建议
2. **公司概况** - 基本信息和主营业务
3. **股权结构** - 股东构成和实际控制人
4. **业务分析** - 商业模式和竞争优势
5. **财务分析** - 财务状况和盈利能力
6. **风险评估** - 各类风险识别和评估
7. **估值分析** - 企业价值评估
8. **投资建议** - 投资结论和注意事项

## 🎯 适用场景

- 🔍 投资机构项目筛选
- 💰 贷款前企业评估
- 🤝 企业并购尽职调查
- 📈 股权投资决策
